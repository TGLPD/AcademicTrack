AcademicTrack - Student Grade & GPA Tracker

Overview

AcademicTrack is a Java-based console application designed to help students and administrators manage academic records. It facilitates the tracking of course grades, calculation of GPA/CGPA, and generation of academic reports.

Features

User Management: Register new students with unique IDs.

Course Enrollment: dynamic addition of courses and credits.

Grade Processing: Converts letter grades (A, B, C...) to grade points automatically.

Analytics: Calculates GPA weighted by course credits.

Reporting: Displays a formatted transcript of all completed courses.

Technologies Used

Language: Java (JDK 17+)

Paradigm: Object-Oriented Programming (OOP)

Tools: VS Code / IntelliJ IDEA, Git

Steps to Install & Run

Clone the repository:

git clone [https://github.com/yourusername/AcademicTrack.git](https://github.com/yourusername/AcademicTrack.git)


Compile the code:
Navigate to the src folder and run:

javac App.java


Run the application:

java App


Usage Instructions

Select Option 1 to Register a new student.

Select Option 2 to Add a course and grade to a student's record.

Select Option 3 to Generate a GPA report/transcript.
